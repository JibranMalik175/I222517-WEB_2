Lab Task 3: Responsive Navigation with Theme Switcher
Duration: 30 minutes | Marks: 30

Create a responsive navigation bar that transforms into a hamburger menu on mobile devices and
includes a dark/light mode toggle switch. The page should have a hero section with content to
demonstrate the theme changes.
Requirements Breakdown
1. HTML Structure
• Navigation Bar: Create a header containing:
o Logo/Brand name on the left
o Navigation links (Home, About, Services, Contact) in the center/right
o Theme toggle button/icon on the far right
o Hamburger icon (visible only on mobile)
• Hero Section: Add a heading, paragraph, and a call-to-action button below the navbar to
demonstrate the theme styling
2. CSS Styling & Responsiveness
• Desktop View (>768px):
o Horizontal navbar with all links visible
o Theme toggle button styled prominently
o Smooth hover effects on links and button
• Mobile View (<768px):
o Hide the navigation links initially
o Show the hamburger icon (☰) on the right side (left of theme toggle)
o When hamburger is clicked, show navigation links in a vertical dropdown/menu
o Menu should overlay or push down content
• Theme Styles:
o Light Theme: Light background (e.g., #f8f9fa), dark text, white navbar
o Dark Theme: Dark background (e.g., #222), light text, darker navbar
o Use CSS variables for easy theme switching

3. JavaScript Functionality

• Hamburger Menu: Toggle the visibility of navigation links when hamburger icon is
clicked on mobile
• Theme Switcher:
o Toggle between light and dark themes when the theme button is clicked
o Change navbar background, text colors, and hero section colors accordingly
o Update button icon/text (e.g., "" for light mode, " " for dark mode)
• Responsive Behavior: Ensure hamburger menu hides/reshows properly when resizing
window
• Add smooth transitions for theme changes and menu animations
• Save the user's theme preference in localStorage
Sample Interaction Flow
1. On desktop: User sees full navbar with theme toggle
2. On mobile: User sees logo, hamburger icon, and theme toggle
3. User clicks hamburger → navigation links slide down
4. User clicks theme toggle → entire site switches between light and dark mode
5. Resize window → navigation adapts automatically
